# パルセーション閾値測定プログラム作成計画

## 1. 実験の目的と概要

* **目的**: ピンクノイズ（マスカー: M）と500Hz正弦波（テスト信号: T）を交番させた刺激音（M-T-M-T-M-T-M）を用い、テスト信号のパルセーション閾値を測定する。
* **測定手法**: Jesteadt (1980) が提案した2系列の適応的測定法（Adaptive Procedure）。
* **独立変数**: テスト信号の微細構造における Interaural Time Delay (ITD)。
* **従属変数**: テスト信号のレベル（パルセーション閾値）。
* **基準レベル**: マスカーの提示レベルは、事前に測定する「1kHz純音の聴取閾値」を0 dB SLとし、そこから +60 dB SL とした値に固定する。

## 2. 開発・実行環境

* **ソフトウェア**: PsychoPy (CoderインターフェースでPythonスクリプトを直接記述)
* **実行環境**: Antigravity
* **音声出力**: ヘッドホン聴取（両耳提示・ステレオ出力）

## 3. 実験プログラムの全体フロー

実験は大きく2つのフェーズ（Phase 1: 閾値測定、Phase 2: 本実験）で構成し、1つのスクリプト内で連続して実行、または別スクリプトにして設定ファイルで変数を引き継ぐ形にします。

1. **初期ダイアログ**: 被験者IDの入力、テストするITDリストの入力（例: `0, 200, 400`）。
2. **Phase 1: 1kHz聴取閾値の測定（SL基準の取得）**
* 1kHzの純音を用いて、被験者の最小可聴値を測定する。
* 測定結果（dB FS: フルスケールに対するdB値など）を `0 dB SL` の基準値として保存。


3. **Phase 2: パルセーション閾値の測定（本実験）**
* Phase 1で得た基準値に基づき、マスカーのレベルを計算（基準値 + 60 dB）。
* 入力されたITD条件をランダムな順序で提示。
* 各ITD条件について、Jesteadt法によるパルセーション閾値のトラッキングを実施。


4. **データ保存・終了**: トライアルごとの応答データと、算出された閾値をCSVに書き出し。

## 4. 各モジュールの詳細設計

### 4.1. Phase 1: SL基準測定モジュール (1kHz閾値)

PsychoPyに組み込まれている階段法（`data.StairHandler`）を利用してシンプルに実装します。

* **刺激音**: 1kHz純音（長さ500ms程度、両端20msコサインテーパー、ITD=0の両耳提示）。
* **タスク**: Yes/No課題（「音が聞こえたらキーを押す」）。
* **トラッキング**: 1-down 1-up（または 2-down 1-up）の単純な階段法を用い、数回の反転（Reversal）後に終了。
* **出力**: 最終的な閾値レベル（システム上のdB値）。これを本実験の基準値とする。

### 4.2. Phase 2: 刺激音生成モジュール

NumPyを用いて波形データを動的に配列として合成し、PsychoPyの `sound.Sound` オブジェクトに渡します。

* **構成要素（MとT）**:
* 各要素の長さ: 165 ms
* 立ち上がり/立ち下がり: 20 msのコサインテーパー


* **マスカー (M)**: ピンクノイズ。ITD = 0。レベルは `Phase 1の閾値 + 60 dB`。
* **テスト信号 (T)**: 500Hz正弦波。
* **微細構造 (Fine Structure)**: 指定されたITD（秒）分の時間差（位相差）を左右チャネルに付与する。
* **エンベロープ (Envelope)**: ITD = 0（左右同時に立ち上がり、立ち下がる）。


* **交番結合**: M-T-M-T-M-T-M（計7要素）。隣り合う要素同士は 20 ms オーバーラップ（クロスフェード）させる。全体の長さは 1035 ms。

### 4.3. Phase 2: Jesteadt (1980) 適応的測定アルゴリズム

2つの独立した系列（Track A / Track B）を交番して提示するカスタムクラスを作成します。被験者の回答は「連続 (Continuous)」か「断続 (Pulsating)」の2択です。

* **Track A（高レベル開始）**
* 開始時: 明らかに「断続」して聞こえる高いレベル。
* 目的: **「断続」と71%の確率で回答されるレベル**の探索。
* ルール: 2回連続で「断続」と回答 → レベルを下げる。1回でも「連続」と回答 → レベルを上げる。


* **Track B（低レベル開始）**
* 開始時: 明らかにマスキングされて「連続」して聞こえる低いレベル。
* 目的: **「連続」と71%の確率で回答されるレベル**の探索。
* ルール: 2回連続で「連続」と回答 → レベルを上げる。1回でも「断続」と回答 → レベルを下げる。


* **インターリーブ（交差）**: 毎試行、未終了のTrack AとBからランダムに1つ選んで提示する。被験者にはどちらのTrackかは分からない。
* **ステップサイズと終了条件**:
* 初期ステップ幅は大きめ（例: 2 dB）、2〜3回の反転で小さくする（例: 1 dB）。
* 各Trackが規定の反転回数（例: 8回）に達したらそのTrackは終了。両方終了で次のITD条件へ。
